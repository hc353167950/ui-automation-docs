🚀 安装配置指南

详细的安装步骤和环境配置指南

⚡快速安装

1

克隆项目

# 克隆项目仓库
git clone https://github.com/hc353167950/new_project.git
cd new_project
2

安装Python依赖

# 创建虚拟环境（推荐）
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# 安装依赖
pip install -r requirements.txt
3

安装Node.js依赖

# 进入MidScene目录
cd midscene_integration

# 安装Node.js依赖
npm install

# 返回项目根目录
cd ..
4

配置环境变量

# 复制环境变量模板
cp .env.example .env

# 编辑.env文件，填入API密钥
# OPENAI_API_KEY=your_api_key_here
# ANDROID_DEVICE_ID=your_device_id_here

🔌MCP服务配置

Claude Desktop配置

编辑配置文件：%APPDATA%\Claude\claude_desktop_config.json

{
  "mcpServers": {
    "appium-automation": {
      "command": "python",
      "args": ["appium_mcp/main.py"]
    }
  }
}

PyCharm AI Assistant配置

在PyCharm中：Settings → Tools → AI Assistant → Model Context Protocol (MCP)

Name: Appium MCP Server
Command: python
Args: appium_mcp/main.py
